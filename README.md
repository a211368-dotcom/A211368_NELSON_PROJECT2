# Project 2 - Enhanced System

##  Project Description
This project is a web-based e-Portfolio / enhanced system developed as part of academic coursework. It demonstrates structured UI design using HTML and CSS, including navigation, project sections, and responsive layout elements.

The system is designed to present lab works and projects in an organized and user-friendly manner.

---

##  SDG Theme
This project aligns with:

**SDG 4: Quality Education**

It supports better learning experiences by providing:
- Organized digital learning materials
- Easy access to lab and project documentation
- Improved presentation of academic work

---

##  Features
- Home page with navigation menu  
- Lab section to display practical works  
- Project section for enhanced system showcase  
- Clean and responsive UI design  
  - Easy navigation between pages (Home, Labs, Projects)  
-  Card-based layout for better content display
